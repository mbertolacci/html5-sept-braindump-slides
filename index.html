  # Know x is defined

Touch events

  • They're not the same as mouse events.
    • Eg click - 200ms delay
    • There's no hover on touch
  • You should listen for touch style events directly, but it's a bit complex, so...

You really want to use HammerJS

No dependencies, but can optionally be used with jQuery.

Also listens for the equivalent mouse events

Hammer(element).on('tap', function() {
  console.log('You touched this');
})
.on('swipeleft', function() {
  console.log('You swiped this');
})
